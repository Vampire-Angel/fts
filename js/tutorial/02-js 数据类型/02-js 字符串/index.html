字符串

  • 字符串可采用new String()来定义，采用toString()输出(不推荐)，而推荐使用简写：const name='Miracle'
  • 字符串内部如果存在特殊字符(如'，", \等)，需要采用\来进行转义
  • 多个字符串采用'+'进行拼接(不推荐)，而推荐采用`(模板字符串)来输出
  • 模板字符串内部不仅可为变量，还可为表达式或函数，甚至还支持嵌套
  • 使用\t输出制表符，使用\n输出回车符，使用&nbsp;在html中输出空白符
  • "+"和"`"区别：前者有时需加转义字符(如\n),后者直接输出即可(如多行)
  • 推荐大多数情况下使用''(单引号)来声明字符串，特殊情况下(如字符串内部，或JSON)才使用""(双引号)
