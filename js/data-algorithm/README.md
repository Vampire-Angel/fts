> [前端数据结构集锦](https://github.com/miracle-git/fts/tree/master/js/data-algorithm)。

## 运行步骤
- 打开data-algorithm项目安装依赖包
```bash
npm install
```
- 打开每一个数据结构示例
```bash
webpack              // 打包示例
webpack-dev-server   // 本地运行
```
- 如果有些示例包含webpack.demo.config.js
```bash
webpack-dev-server --config webpack.demo.config.js
```
- 接下来，就开始学习前端算法把!
